# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-23

### Added

- Initial repository structure for `lighthouse-skill-pack`
- Five production-ready skills:
  - `lighthouse-core`
  - `lighthouse-static-html`
  - `lighthouse-react-vite`
  - `lighthouse-nextjs`
  - `lighthouse-wordpress`
- Installer scripts for Claude Code, Codex, and PowerShell
- Static website with skill browsing, markdown viewing, copy, and download
- Playwright E2E test suite with 85%+ coverage threshold gate for `website/app.js`
- Examples folder with per-stack usage notes and Lighthouse CI workflow sample
- Contribution guide and GitHub-ready docs
